* Python 3
* Git

Background:

* Limited experience building agent systems

Goals:

* Understand how agents are constructed and deployed

Rules:

* Explain architectural decisions.
* Prefer simple solutions.

* Show me the underlying code and concepts.
* Assume I am learning and want to understand why things are built a certain way.
* When creating files, explain their purpose.
* Suggest incremental steps rather than large complex systems.

Current Objective:
Build a minimal agent in Python that demonstrates:

1. Receiving a request
2. Reasoning about what to do
3. Producing an output
4. Saving results to disk

The goal is understanding the agent pattern rather than building a production system.
